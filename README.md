## 📚 Project List

### 1. Geolocation Tracker 🌍
A Python script that retrieves the user's geolocation using their IP address and displays it on an interactive map.

**Technologies Used**
- Python
- Requests
- Folium

**Features**
- Detects IP-based location
- Displays latitude and longitude
- Generates a map with a location marker
